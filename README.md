# sweetPopUp
Easy popup generator
```javascript
$.sweetPopUp({
	imageUrl: "",
	link: "",
	target: "_blank",
	animateIn: "fadeIn",
	animateOut: "fadeOut",
	oneTime: true,
	positionX: "top", //top, middle, bottom
	positionY: "center", // left, center, right
	gap: 100,
	start: {
		day: 10,
		month: 10,
		year: 2015,
		hour: 10,
		minute: 10
	},
	end: {
		day: 10,
		month: 10,
		year: 2015,
		hour: 10,
		minute: 10
	}
});
```
